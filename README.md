## Software_Engineering_Phase_1

## Project Name :- Sentiment Analysis of Voice

## Description  :- 
  * Software interface will take input as an Audio file.
  * And output will be the emotions which will be categorized considering the conversation in the file.
  * Audio in the file should be in English only
  * Updated version may contain multilingual audio analysis.
  
## Tools and Technologies :- 
  * Machine Learning(Python, TensorFlow)
  * Django Framework for building web based application
  
## Comparison with the state of art :-
 * One of the AI and Machine Learning product on google cloud platform is "Cloud Speech-to-Text"
 * It makes user to input an audio either by microphone or file upload
 * It recognizes nearly 120 languages and variants
 * Moreover interface asks about speaker diarization which is still in the BETA version. User needs to give number of speakers which are involved in converstion
 * Currently this is state of art product in the field of Speech Sentiment Analysis
